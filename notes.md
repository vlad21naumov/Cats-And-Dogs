# Jupyter server

```
pip install ipykernel
```

```
pip install notebook  # for Jupyter Notebook
pip install jupyterlab  # for JupyterLab
```

```
python -m ipykernel install --user --name=myenv2 --display-name "Python (myenv2)"
```

To verify kernels:

```
jupyter kernelspec list
```
